[#overview]

[float]
=== Beschreibung
Konvertiert einen Wert in den link:../../data-types/char[char]-Datentyp.
[%hardbreaks]


[float]
=== Syntax
`char(x)` +
`(char)x` (Typ-Konvertierung wie in C)


[float]
=== Parameter
`x`: Ein Wert beliebigen Typs. Erlaubte Datentypen: Beliebiger Datentyp.


[float]
=== Rückgabewert
Datentyp: `char`.
