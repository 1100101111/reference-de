= byte()


// ÜBERSICHTSABSCHNITT STARTET
[#overview]
--

[float]
=== Beschreibung
Konvertiert einen Wert in den `link:../../data-types/byte[byte]`-Datentyp.
[%hardbreaks]


[float]
=== Syntax
`byte(x)` +
`(byte)x` (Typ-Konvertierung wie in C)


[float]
=== Parameter
`x`: Ein Wert beliebigen Typs. Erlaubte Datentypen: Beliebiger Datentyp.


[float]
=== Rückgabewert
Datentyp: `byte`.
