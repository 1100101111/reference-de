= word

// ÜBERSICHTSABSCHNITT STARTET
[#overview]
--

[float]
=== Beschreibung
Ein Wort kann eine vorzeichenlose Zahl von mindestens 16 Bits (von 0 bis 65535) speichern.
[%hardbreaks]


[float]
=== Syntax
`word var = val`


[float]
=== Parameter
`var`: Variablennamen. +
`val`: Der Wert, der dieser Variablen zugewiesen werden soll.
