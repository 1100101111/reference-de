= tan(rad)


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Beschreibung
Berechnet den Tangens eines Winkels (Im Bogenmaß). Das Ergebnis liegt zwischen undendlich und negativ unendlich.
[%hardbreaks]


[float]
=== Syntax
`tan(rad)`


[float]
=== Parameter
`rad`: Der Winkel im Bogenmaß (`float`).

[float]
=== Rückgabewert
Den Tangens des Winkels (`double`).
