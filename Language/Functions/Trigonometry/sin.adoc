= sin(rad)


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Beschreibung
Berechnet den Sinus eines Winkels (Im Bogenmaß). Das Ergebnis liegt zwischen -1 und 1.
[%hardbreaks]


[float]
=== Syntax
`sin(rad)`


[float]
=== Parameter
`rad`: Der Winkel im Bogenmaß (`float`).

[float]
=== Gibt zurück
Den Sinus des Winkels (`double`).
