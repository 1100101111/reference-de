= highByte()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Beschreibung
Liest das most-significant (linkeste) Bit eines Wortes (oder das zweitkleinste Bit eines größeren Datentypes).
[%hardbreaks]


[float]
=== Syntax
`highByte(x)`


[float]
=== Parameter
`x`: Wert von beliebigen Typ

[float]
=== Gibt zurück
Das geforderte Byte (byte)
