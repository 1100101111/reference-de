= lowByte()


// ÜBERSICHTSABSCHNITT STARTET
[#overview]
--

[float]
=== Beschreibung
Liest das least-significant (rechteste) Bit eines Wortes oder größeren Datentypes.
[%hardbreaks]


[float]
=== Syntax
`lowByte(x)`


[float]
=== Parameter
`x`: Wert von beliebigen Typ.


[float]
=== Rückgabewert
Das geforderte Byte. Datentyp: `byte`.
